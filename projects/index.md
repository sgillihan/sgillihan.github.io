## [Pburst Pressure Calculation Checker](/projects/pburst-pressure-calculation-checker/)
A burst pressure calculation application that uses ILI vendor spreadsheets to check the ILI vendor reported pressure against a calculated pressure based on reported parameters. 

## [Proxy Data Research: Socioeconomic Impacts](/projects/proxy-data-research/)
Exploring the technical and societal impacts of proxy data use in algorithms and AI/ML systems. Investigates how personal data is collected, what parameters are used as proxies in decision-making algorithms, and the possibility of discrimination throughout the process.

## [Fenwick Tree Sports Tracker](/projects/fenwick-tree-sports-tracker/)
An early school project for my data structures class. Written in C++. Demonstrates the strengths of the Fenwick Tree (Binary Indexed Tree) data structure.

## [One Handy Katie](/projects/one-handy-katie/)
A website for a local businesswoman. Built with Astro, Sanity CMS, and deployed on Netlify. First project building a product for a "client" with a focus on easy long-term maintenance for a non-technical person.
