# Fenwick Tree Sports Tracker

## About This Project
An early school project for my data structures class. Written in C++.

This project implements a Fenwick Tree (Binary Indexed Tree) data structure to efficiently track and query sports statistics. Activity entries can be mass imported from CSV, and individual entries can also be manually added, edited, and deleted. The Fenwick Tree structure efficiently rebuilds the trees and allows summary statistics to be updated quickly. Watch the project walk-through video linked in the Repository README for more information.
